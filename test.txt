## 3. THE CORE INNOVATION — CLIENT-SIDE SMART SAMPLING

This is what makes the cost model work. Read this section before writing any code.

### Problem with naive approach
Sending all game events to the server for 1M users at 1,000 events/user/day =
1 billion events/day. Processing this costs ~$12,000/mo minimum. Unacceptable.

### Solution: 3-tier event filtering

```
TIER 1 — Client-side SDK (runs in-process, zero server cost)
  Every user: compute rolling RiskScore on last 30 seconds of behavior
  RiskScore < 0.3  →  drop event, accumulate into session summary only
  RiskScore ≥ 0.3  →  enter TIER 2

TIER 2 — Anomaly burst (triggered by SDK, real-time server path)
  Stream full events for this session window to gateway
  Server runs Isolation Forest + rule engine → fast verdict
  Verdict < 0.7  →  back to TIER 1 monitoring
  Verdict ≥ 0.7  →  enter TIER 3

TIER 3 — Deep ML analysis
  XGBoost + LSTM on full session history
  Produces final ensemble score → decision engine
```

### Session summary batching (the traffic killer)
Clean players (TIER 1 only) send ONE compressed summary per 5-minute session window:
- 32 pre-computed feature floats (aim stats, movement stats, timing stats)
- gzip compressed → ~200 bytes per batch
- Sent via HTTP POST (not WebSocket) to reduce connection overhead

At 1M users, 95% clean, 288 batches/day each:
- Clean traffic: 950,000 × 288 × 200 bytes = **54 GB/day** (vs 1TB+ for full events)
- Suspicious traffic: 50,000 × full events = **~5 GB/day**
- **Total: ~59 GB/day** — Hetzner charges €1/TB. This costs **€1.77/mo** in bandwidth.

### SDK local RiskScore computation
```python
# sdk/python/anticheat_sdk/local_scorer.py
import math
from collections import deque
from dataclasses import dataclass, field
from typing import Deque

@dataclass
class LocalScorer:
    """
    Runs entirely in-process on game client/server.
    No network calls. No dependencies beyond stdlib + numpy.
    """
    window_seconds: int = 30
    _shot_intervals: Deque[float] = field(default_factory=lambda: deque(maxlen=100))
    _aim_deltas: Deque[tuple] = field(default_factory=lambda: deque(maxlen=200))
    _move_speeds: Deque[float] = field(default_factory=lambda: deque(maxlen=100))
    _last_shot_ts: float = 0.0

    def on_shot_fired(self, timestamp_ms: float, aim_vector: tuple) -> None:
        if self._last_shot_ts > 0:
            interval = timestamp_ms - self._last_shot_ts
            self._shot_intervals.append(interval)
        self._last_shot_ts = timestamp_ms

    def on_mouse_move(self, dx: float, dy: float) -> None:
        self._aim_deltas.append((dx, dy))

    def on_move(self, speed: float) -> None:
        self._move_speeds.append(speed)

    def compute_risk_score(self) -> float:
        """Returns 0.0 (clean) to 1.0 (highly suspicious). No network."""
        scores = []

        # Trigger bot signal: inhuman consistency in shot timing
        if len(self._shot_intervals) >= 10:
            mean = sum(self._shot_intervals) / len(self._shot_intervals)
            if mean > 0:
                cv = (
                    math.sqrt(sum((x - mean)**2 for x in self._shot_intervals)
                    / len(self._shot_intervals)) / mean
                )
                # Human CV ~0.15-0.4, trigger bot CV < 0.02
                scores.append(max(0.0, 1.0 - (cv / 0.05)))

        # Aimbot signal: impossible angular velocity
        if len(self._aim_deltas) >= 20:
            magnitudes = [math.sqrt(dx**2 + dy**2) for dx, dy in self._aim_deltas]
            max_snap = max(magnitudes)
            # >3000 deg/sec equivalent is physically impossible for humans
            if max_snap > 3000:
                scores.append(min(1.0, max_snap / 5000))

        # Speed hack signal: movement speed anomaly
        if len(self._move_speeds) >= 10:
            max_speed = max(self._move_speeds)
            # Game-specific: inject MAX_LEGITIMATE_SPEED via SDK config
            max_legit = self._config.get("max_speed", 800.0)
            if max_speed > max_legit * 1.05:
                scores.append(min(1.0, (max_speed - max_legit) / max_legit))

        return max(scores) if scores else 0.0
```

## 5. FEATURE ENGINEERING — UNIVERSAL BEHAVIORAL FEATURES

These are computed server-side from raw events. They are game-agnostic by design.
All feature computation lives in `models/features/compute.py` — never duplicate this code.

```python
# models/features/compute.py
import numpy as np
from scipy.stats import entropy
from typing import Sequence
from .schema import PlayerEvent, SummaryFeatures

def compute_aim_features(events: Sequence[PlayerEvent]) -> dict:
    shots = [e for e in events if e.event_type == "shot_fired"]
    hits  = [e for e in events if e.event_type == "hit_registered"]
    frames = [e for e in events if e.event_type == "input_frame"]

    features = {}

    # --- Trigger interval coefficient of variation ---
    if len(shots) >= 10:
        intervals = np.diff([s.timestamp_ms for s in shots])
        cv = intervals.std() / intervals.mean() if intervals.mean() > 0 else 0
        features["trigger_interval_cv"] = float(cv)
        # Human range: 0.15–0.5 | Trigger bot: < 0.03

    # --- Pre-shot micro-correction ---
    # Measures aim vector change in last 80ms before shot. Aimbots don't correct.
    if shots and frames:
        corrections = []
        for shot in shots:
            recent = [f for f in frames
                      if shot.timestamp_ms - 80 <= f.timestamp_ms < shot.timestamp_ms]
            if recent:
                deltas = [f.payload.get("mouse_delta", [0,0]) for f in recent]
                mag = np.sqrt(np.array(deltas)**2).sum(axis=1).mean()
                corrections.append(float(mag))
        if corrections:
            features["pre_shot_micro_correction"] = float(np.mean(corrections))

    # --- Snap angular velocity (p95) ---
    if frames:
        deltas = np.array([f.payload.get("mouse_delta", [0,0]) for f in frames])
        dt = np.array([f.payload.get("frame_ms", 16) for f in frames])
        velocities = np.linalg.norm(deltas, axis=1) / np.maximum(dt, 1) * 1000
        features["snap_velocity_p95"] = float(np.percentile(velocities, 95))

    # --- Recoil pattern entropy ---
    if len(shots) >= 20:
        recoil_vecs = [s.payload.get("aim_vector", [0,0,0]) for s in shots]
        compensations = np.diff(recoil_vecs, axis=0)
        hist, _ = np.histogram(compensations[:,0], bins=20, density=True)
        features["recoil_entropy"] = float(entropy(hist + 1e-10))
        # Low entropy = bot (same pattern), high entropy = human

    # --- Headshot rate ---
    head_hits = sum(1 for h in hits if h.payload.get("hit_zone") == "head")
    features["headshot_rate"] = head_hits / max(len(hits), 1)

    return features


def compute_movement_features(events: Sequence[PlayerEvent]) -> dict:
    moves = [e for e in events if e.event_type == "player_moved"]
    features = {}

    if len(moves) < 10:
        return features

    speeds = np.array([m.payload.get("speed", 0.0) for m in moves])
    features["avg_speed"]  = float(speeds.mean())
    features["max_speed"]  = float(speeds.max())

    # --- Movement entropy (teleport hack = zero entropy) ---
    positions = np.array([m.payload.get("position", [0,0,0]) for m in moves])
    displacements = np.diff(positions, axis=0)
    mags = np.linalg.norm(displacements, axis=1)
    hist, _ = np.histogram(mags, bins=20, density=True)
    features["movement_entropy"] = float(entropy(hist + 1e-10))

    # --- Strafe period detection (bots strafe with mechanical rhythm) ---
    vx = np.array([m.payload.get("velocity", [0,0,0])[0] for m in moves])
    if len(vx) >= 32:
        fft = np.abs(np.fft.rfft(vx))
        dominant_freq_idx = np.argmax(fft[1:]) + 1
        features["strafe_period_detected"] = float(dominant_freq_idx)

    # --- Path curvature variance ---
    if len(positions) >= 3:
        v1 = np.diff(positions[:-1], axis=0)
        v2 = np.diff(positions[1:],  axis=0)
        norms = np.linalg.norm(v1, axis=1) * np.linalg.norm(v2, axis=1)
        safe_norms = np.maximum(norms, 1e-10)
        dots = (v1 * v2).sum(axis=1) / safe_norms
        curvatures = np.arccos(np.clip(dots, -1, 1))
        features["path_curvature_var"] = float(curvatures.var())

    return features


def compute_timing_features(events: Sequence[PlayerEvent]) -> dict:
    """Reaction time, burst rhythm, wallhack proxy."""
    features = {}
    shots = [e for e in events if e.event_type == "shot_fired"]
    kills = [e for e in events if e.event_type == "hit_registered"
             and e.payload.get("hit_zone") == "head"]

    # --- Burst rhythm autocorrelation ---
    if len(shots) >= 20:
        ts = np.array([s.timestamp_ms for s in shots], dtype=float)
        intervals = np.diff(ts)
        if len(intervals) >= 10:
            autocorr = np.correlate(intervals, intervals, mode='full')
            mid = len(autocorr) // 2
            lag1 = autocorr[mid + 1] / autocorr[mid] if autocorr[mid] != 0 else 0
            features["burst_autocorr"] = float(lag1)
            # High autocorr at lag-1 = robot-like repetition

    return features


def build_feature_vector(events: Sequence[PlayerEvent],
                         summary: "SummaryFeatures | None" = None) -> np.ndarray:
    """
    Canonical 48-dim feature vector used by ALL models.
    If summary provided (TIER 1 batch), use its pre-computed values.
    If raw events provided (TIER 2/3), compute fresh.
    Train and serve MUST call this same function — no divergence.
    """
    if summary:
        return np.array([
            summary.headshot_rate,
            summary.avg_reaction_time_ms,
            summary.snap_velocity_p95,
            summary.aim_jitter_variance,
            summary.pre_shot_correction_avg,
            summary.recoil_entropy,
            summary.trigger_interval_cv,
            summary.burst_autocorr,
            float(summary.shots_fired),
            float(summary.hits_registered),
            summary.avg_speed,
            summary.max_speed,
            summary.movement_entropy,
            summary.strafe_period_detected,
            summary.path_curvature_var,
            float(summary.kills),
            float(summary.deaths),
            float(summary.playtime_seconds),
            *([0.0] * 30)   # padding to 48 dims
        ], dtype=np.float32)

    feats = {}
    feats.update(compute_aim_features(events))
    feats.update(compute_movement_features(events))
    feats.update(compute_timing_features(events))

    FEATURE_ORDER = [
        "headshot_rate", "snap_velocity_p95", "aim_jitter_variance",
        "pre_shot_micro_correction", "recoil_entropy", "trigger_interval_cv",
        "burst_autocorr", "movement_entropy", "avg_speed", "max_speed",
        "path_curvature_var", "strafe_period_detected",
    ]
    return np.array([feats.get(k, 0.0) for k in FEATURE_ORDER]
                    + [0.0] * (48 - len(FEATURE_ORDER)), dtype=np.float32)
```

@app.websocket("/v1/stream/{player_id}")
async def event_stream(ws: WebSocket, player_id: str):
    """TIER 2: real-time event stream for suspicious sessions only."""
    await ws.accept()
    try:
        async for raw in ws.iter_bytes():
            event = PlayerEvent.model_validate_json(raw)
            # Verify checksum
            expected = hashlib.sha256(
                f"{event.event_id}{event.timestamp_ms}"
                f"{json.dumps(event.payload, sort_keys=True)}".encode()
            ).hexdigest()
            if event.client_checksum != expected:
                await ws.close(code=4003, reason="checksum_mismatch")
                return
            await app.state.producer.send(
                "realtime-events",
                key=player_id.encode(),
                value=raw,
            )
    except WebSocketDisconnect:
        pass

@app.get("/health")
async def health():
    return {"status": "ok", "version": settings.VERSION}

### 6.2 Detector service (`services/detector/`)

```python
# services/detector/consumer.py
"""
Real-time rule engine + Isolation Forest.
Consumes from 'realtime-events' Kafka topic.
SLA: process each event in <50ms end-to-end.
"""
from aiokafka import AIOKafkaConsumer, AIOKafkaProducer
import asyncio, joblib, numpy as np
from models.features.schema import PlayerEvent
from models.features.compute import build_feature_vector
from .rules import evaluate_rules
from .session_cache import SessionCache

RULES = [
    # (name, condition_fn, score)
    ("impossible_headshot",  lambda e: (
        e.event_type == "hit_registered"
        and e.payload.get("hit_zone") == "head"
        and e.payload.get("distance_m", 0) > 80
    ), 0.85),
    ("speed_hack", lambda e: (
        e.event_type == "player_moved"
        and e.payload.get("speed", 0) > settings.MAX_SPEED * 1.08
    ), 0.90),
    ("trigger_interval_bot", lambda e: (
        e.event_type == "shot_fired"
        and e.payload.get("trigger_interval_ms", 999) < 8
    ), 0.95),
    ("wallhack_proxy", lambda e: (
        e.event_type == "hit_registered"
        and not e.payload.get("server_validated", True)
    ), 0.80),
]

async def run():
    consumer = AIOKafkaConsumer(
        "realtime-events",
        bootstrap_servers=settings.REDPANDA_BROKERS,
        group_id="detector-group",
        auto_offset_reset="latest",
    )
    producer = AIOKafkaProducer(bootstrap_servers=settings.REDPANDA_BROKERS)
    iso_forest = joblib.load(settings.ISO_FOREST_PATH)
    session_cache = SessionCache(redis_url=settings.REDIS_URL)

    await asyncio.gather(consumer.start(), producer.start())

    async for msg in consumer:
        event = PlayerEvent.model_validate_json(msg.value)
        t0 = asyncio.get_event_loop().time()

        # 1. Rule check — instant
        rule_score = 0.0
        triggered_rule = None
        for name, condition, score in RULES:
            if condition(event):
                rule_score = max(rule_score, score)
                triggered_rule = name

        # 2. Isolation Forest on rolling window
        window = await session_cache.get_window(event.session_id, event)
        if len(window) >= 10:
            fvec = build_feature_vector(window).reshape(1, -1)
            if_score = float(-iso_forest.score_samples(fvec)[0])  # higher = more anomalous
        else:
            if_score = 0.0

        combined = max(rule_score, if_score * 0.7 + event.local_risk * 0.3)

        # 3. Route decision
        result = {
            "player_id": event.player_id,
            "session_id": event.session_id,
            "game_id":    event.game_id,
            "score":      combined,
            "rule":       triggered_rule,
            "if_score":   if_score,
            "latency_ms": (asyncio.get_event_loop().time() - t0) * 1000,
        }

        if combined >= 0.5:
            await producer.send("detection-results", value=str(result).encode())

        # Alert if SLA breached
        if result["latency_ms"] > 50:
            await producer.send("sla-breaches", value=str(result).encode())
```

### 6.3 ML Worker (`services/ml_worker/`)

```python
# services/ml_worker/consumer.py
"""
Async XGBoost + LSTM inference.
Consumes 'session-summaries' (TIER 1) and 'detection-results' (TIER 2 escalations).
No SLA constraint — quality over speed.
"""
import asyncio, torch, xgboost as xgb, numpy as np
from aiokafka import AIOKafkaConsumer, AIOKafkaProducer
from models.features.schema import SessionSummary
from models.features.compute import build_feature_vector
from .lstm_model import SessionLSTM
from .model_loader import ModelLoader

async def run():
    loader = ModelLoader(mlflow_uri=settings.MLFLOW_URI)
    xgb_model: xgb.XGBClassifier = await loader.load("xgboost_cheat_v2")
    lstm_model: SessionLSTM       = await loader.load("lstm_session_v1")

    consumer = AIOKafkaConsumer(
        "session-summaries", "detection-results",
        bootstrap_servers=settings.REDPANDA_BROKERS,
        group_id="ml-worker-group",
    )
    producer = AIOKafkaProducer(bootstrap_servers=settings.REDPANDA_BROKERS)
    await asyncio.gather(consumer.start(), producer.start())

    async for msg in consumer:
        if msg.topic == "session-summaries":
            summary = SessionSummary.model_validate_json(msg.value)
            fvec = build_feature_vector([], summary=summary.features).reshape(1, -1)
        else:
            # Escalation from detector — fetch cached events
            data = eval(msg.value.decode())
            fvec = np.zeros((1, 48), dtype=np.float32)  # placeholder

        xgb_prob  = float(xgb_model.predict_proba(fvec)[0][1])
        lstm_prob  = float(torch.sigmoid(lstm_model(
            torch.tensor(fvec).unsqueeze(0)
        )).item())

        ensemble = (xgb_prob * 0.6) + (lstm_prob * 0.4)

        await producer.send("ml-scores", value=str({
            "player_id": data.get("player_id", summary.player_id if msg.topic == "session-summaries" else ""),
            "xgb_score":  xgb_prob,
            "lstm_score": lstm_prob,
            "ensemble":   ensemble,
        }).encode())
```

### 6.4 Decision Engine (`services/decision/`)

```python
# services/decision/consumer.py
"""
Combines detector scores + ML scores → final verdict.
Routes to action queue or human review queue.
"""
THRESHOLDS = {
    "auto_shadow_ban":  0.95,   # immediate action, queue for human confirmation
    "auto_flag":        0.80,   # restrict matchmaking, add to review
    "escalate_ml":      0.50,   # send to ml_worker for deep analysis
    "monitor":          0.30,   # increase sampling rate
    "clear":            0.15,   # log as clean negative training example
}

# False positive mitigations
FP_MITIGATIONS = [
    # If player has >200 hours clean history, require higher confidence
    lambda ctx: ctx["score"] * 0.7 if ctx.get("clean_hours", 0) > 200 else ctx["score"],
    # If detector and ML disagree by >0.35, route to human review
    lambda ctx: 0.79 if abs(ctx.get("detector_score", 0)
                           - ctx.get("ml_score", 0)) > 0.35 else ctx["score"],
    # Never auto-ban on first session — must have history
    lambda ctx: min(ctx["score"], 0.79) if ctx.get("session_count", 0) < 3 else ctx["score"],
]

async def make_decision(ctx: dict) -> dict:
    score = ctx["score"]
    for mitigation in FP_MITIGATIONS:
        score = mitigation({**ctx, "score": score})

    if score >= THRESHOLDS["auto_shadow_ban"]:
        action = "shadow_ban"
        queue  = "action-queue"
    elif score >= THRESHOLDS["auto_flag"]:
        action = "flag"
        queue  = "action-queue"
    elif score >= THRESHOLDS["escalate_ml"]:
        action = "review"
        queue  = "review-queue"
    elif score >= THRESHOLDS["monitor"]:
        action = "monitor"
        queue  = "monitoring-queue"
    else:
        action = "clear"
        queue  = "training-negatives"

    return {"action": action, "queue": queue, "final_score": score, **ctx}
```

### XGBoost training script
```python
# models/training/train_xgboost.py
import xgboost as xgb, pandas as pd, numpy as np, mlflow
from sklearn.model_selection import train_test_split
from sklearn.metrics import roc_auc_score, precision_recall_curve

FEATURES = [
    "trigger_interval_cv", "snap_velocity_p95", "headshot_rate",
    "movement_entropy", "recoil_entropy", "pre_shot_micro_correction",
    "burst_autocorr",
]

def train(data_path: str, experiment_name: str = "anticheat-xgboost"):
    df = pd.read_parquet(data_path)
    X, y = df[FEATURES], df["label"]

    X_train, X_test, y_train, y_test = train_test_split(
        X, y, test_size=0.2, stratify=y, random_state=42
    )

    # Class weight for imbalanced data (typically 5% cheat rate)
    scale_pos_weight = (y_train == 0).sum() / (y_train == 1).sum()

    params = {
        "n_estimators":      500,
        "max_depth":         6,
        "learning_rate":     0.05,
        "subsample":         0.8,
        "colsample_bytree":  0.8,
        "scale_pos_weight":  scale_pos_weight,
        "eval_metric":       "auc",
        "early_stopping_rounds": 30,
        "device":            "cpu",   # no GPU needed for this data size
    }

    mlflow.set_experiment(experiment_name)
    with mlflow.start_run():
        model = xgb.XGBClassifier(**params)
        model.fit(X_train, y_train,
                  eval_set=[(X_test, y_test)],
                  verbose=50)

        probs = model.predict_proba(X_test)[:, 1]
        auc = roc_auc_score(y_test, probs)

        # Compute FPR at 0.95 threshold
        precision, recall, thresholds = precision_recall_curve(y_test, probs)
        mask = thresholds >= 0.95
        fpr_at_threshold = 1 - precision[mask][0] if mask.any() else None

        mlflow.log_params(params)
        mlflow.log_metrics({"auc": auc, "fpr_at_095": fpr_at_threshold or 0})
        mlflow.xgboost.log_model(model, "xgboost_cheat")

        print(f"AUC: {auc:.4f} | FPR@0.95: {fpr_at_threshold:.4f}")
        assert auc > 0.85, f"AUC {auc} below minimum threshold 0.85"
        assert fpr_at_threshold < 0.005, f"FPR {fpr_at_threshold} exceeds 0.5% limit"

if __name__ == "__main__":
    train("data/synthetic_v1.parquet")
```

## 12. AGENT RULES — FOLLOW THESE EXACTLY

1. Always run `uv sync` before installing any package. Never use pip directly.
2. All Pydantic models use v2 syntax (`model_validate`, `model_dump`, not `parse_obj`/`dict()`).
3. All database queries are async (asyncpg / SQLAlchemy 2.0 async). No sync DB calls.
4. Feature computation ALWAYS goes through `models/features/compute.py`. Never reimplement.
5. Every service exposes `/health` returning `{"status": "ok", "version": str, "model_version": str}`.
6. Every detection decision is logged with: player_id, feature_hash, all model scores, final decision, threshold used, latency_ms.
7. Thresholds NEVER hardcoded in service code — always read from environment variables with defaults.
8. New cheat pattern = add rule to detector FIRST (deploy in minutes), then add training data, then retrain.
9. When in doubt about a detection, route to human review. Never auto-ban on a single signal.
10. TimescaleDB uses hypertables for all event storage — run `SELECT create_hypertable(...)` in migrations.
11. Redis keys expire — set TTL on all session caches (default: 2 hours after last event).
12. SDK version is a mandatory field on every event — enables gradual rollouts and A/B testing of detection logic.
